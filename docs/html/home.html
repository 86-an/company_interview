面接練習サイト

面接の練習とコツをまとめたサイトです。
制作者の自主学習の側面が強いウェブサイトとなります。
動作はchromeしか行っておりません
パソコン向けです。特に保存機能は。

⚠️ 注意事項

  • 素人が作ったので不具合があるかもしれません。
  • 不具合の修正には時間がかかる場合があります。迅速な対応をお約束することはできません。
  • 現時点では、**Chromeブラウザのみで動作を確認**しています。他のブラウザでは正常に動作しない可能性があります。
  • 深堀質問は「エピソードを詳しく」「なぜそのように考えたか」「どう入社後に活かせるか」の三択かつ**ランダムに出題**されるため、質問内容を考慮していない、**意味の分からない質問になる可能性があります**。
  • 内容は、生成AIによって作成されたものも含まれます。

💡 使い方

  • まずは「コツ」を見て、面接のポイントを把握しましょう。
  • 「ジャンル別」または「模擬面接」を使って、実践的な練習を始めましょう。
  • 最初は質問にスムーズに答えられるように練習し、慣れてきたら「深堀する」ボタンで、より深い質問への対策を行いましょう。

🔗 リンクの説明

Home
このページです。サイトの目的、注意事項、使い方などがまとめられています。
ジャンル別
質問のジャンル（例: 自己紹介、志望動機）やカテゴリ（例: 学生、社会人、共通）を選択して、練習を開始できます。
模擬面接
より実践に近い形で練習できます。対象（学生、社会人）、面接フェーズ（1次、2次）、面接官タイプ（人事、現場、役員など）を選択してスタートします。
質問集
これまでに用意されている全ての質問（全42個）をダウンロードできます。また、ジャンルやカテゴリを選んで質問内容を確認することも可能です。
規約
本サービスの利用規約がまとめられています。Homeの「注意事項」を読めば、基本的な内容は把握できます。
